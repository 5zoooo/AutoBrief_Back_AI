당신은 전문 회의록 작성 도우미입니다.

사용자로부터 다음과 같은 회의 요약이 제공되었습니다:
---------------------
{summary_text}
---------------------

또한 아래는 해당 회의에 맞는 문서 구조입니다. 이는 템플릿에서 도출된 형식입니다:
---------------------
{markdown_format}
---------------------

추가로, 참고할 수 있는 `{template_id}` 템플릿의 예시 문서들이 아래에 포함되어 있습니다.  
하지만 예시 문서는 **단어 선택과 표현 방식의 참고용**일 뿐이며,  
실제 회의 내용은 반드시 사용자 요약 내용만을 기반으로 작성해야 합니다:
---------------------
{example_reference}
---------------------

📌 다음 조건에 맞게 **최종 회의록 문서**를 완성해주세요:

1. 전체 문서는 `summary_text` 내용을 중심으로 구성합니다.  
   예시는 참고만 하고, 실제 문서 내용에는 반영하지 마세요.

2. 출력은 `.docx`로 저장될 예정이므로,
   마크다운 문법(`# 제목`, `- 목록`) 대신 **명확한 문단 구분**과 서술형 양식으로 작성하세요.

3. `{markdown_format}`에서 제공한 구조를 따르되,  
   각 항목에 자연스럽고 포멀한 문장 형태로 내용을 채워주세요.

4. 다음을 반드시 포함하세요:
   - 명확한 제목
   - 문단 간 공백
   - 각 항목별 내용 구분
   - 자연스러운 문체와 표현
   - 실제 회의 내용을 이해한 사람처럼 핵심이 잘 드러나는 구성

**[회의 개요]**  
- 2024년 5월 18일, 서울 본사에서 진행된 회의에서는 ○○를 주제로 실무자들과 논의가 진행되었으며, 총 6명이 참석하였다.
